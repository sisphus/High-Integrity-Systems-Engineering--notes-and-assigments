 Predicates
         Extend propositions with the ability to quantify
       the values of a variable that a proposition is true for

              all: Proposition P(x) holds for all values of x
                                     all x | P[x]
                             all city | Raining[city]
                  all city : AustralianCities | Raining[city]
     some: Proposition P(x) holds for at least one value of x
                                           some x | P[x]
                            some city | not Raining[city]

                                        De Morgan’s Laws

 Relations as Sets
            A relation with arity n (i.e. an n-ary relation) R,
                      is a set of n-tuples of atoms.

     A binary (2-ary) relation R(a,b) is a set of:                                       pairs
     A ternary (3-ary) relation R(a,b,c) is a set of: triples
     A unary (1-ary) relation R(a) is a set of:                                          atoms
                                                                i.e. just an ordinary set
              Sets are relations, and relations are sets.
           Sets are predicates, and predicates are sets.
 Relations are predicates, and predicates are relations.

 Functions as Relations
                 Suppose we have a function f : A → B

                                     f is just a relation

                               i.e. a set of pairs: (a,b)

                                                          Consider some ai : A
 (ai)            (a1,b1)
                 (a2,b2)                  What is f[ai] ?
                 (a3,b3)                  What is ai?         {(ai)}
                    …
                                          f[ai] is equivalent to ai.f
                  (ai,bi)
                    …     i.e. f[ai] = bi which is {(bi)}
